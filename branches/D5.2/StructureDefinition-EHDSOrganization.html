Logical Model: Organization model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSOrganization Version: 0.1.0
Draft as of 2025-08-19 Computable Name: EHDSOrganization

C.3 - EHDS refined base model for Health provider or any other type of organization

Usages:

  • Use this Logical Model: Common model for EHDS document exchange
  • Refer to this Logical Model: Health professional model and Organization model
